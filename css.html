CSS — Presentation and Layout

CSS (Cascading Style Sheets) controls visual presentation. The cascade, specificity, and inheritance determine which rules apply. Modern layout uses Flexbox and Grid; responsive design uses relative units and media queries.

Selectors & Specificity
/* Type, class, id */
p { line-height: 1.6; }
.note { color: #333; }
#highlight { background: #000; color:#fff; }

/* Attribute & pseudo-classes */
input[type="email"]:focus { outline: 2px solid #000; }

/* Specificity tips: prefer classes over ids for reusable styling */
Flexbox & Grid
/* Flexbox */
.container { display:flex; gap:1rem; flex-wrap:wrap; }
.item { flex: 1 1 240px; }

/* Grid */
.grid {
  display:grid;
  grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));
  gap: 1rem;
}

Responsive Techniques

@media (max-width: 768px) {
  .sidebar { display:none; }
  .hero { padding: 1rem; }
}
