JavaScript — Fundamentals & Project

JavaScript adds interactivity. Key concepts include variables and scope, functions and events, DOM manipulation, and storage APIs. The project below is a To‑Do list demonstrating these ideas.

Language Notes
// Variables & scope
let x = 1;        // block-scoped
const PI = 3.14;  // cannot be reassigned

// Functions & events
function greet(name){ return `Hello, ${name}`; }
document.addEventListener('click', () => console.log('clicked'));

// DOM manipulation
const el = document.querySelector('#app');
el.textContent = 'Updated by JS';
Storage & JSON
// Save to localStorage
const data = {{done:false, text:'Task'}}; // pseudo
localStorage.setItem('todos', JSON.stringify([data]));

// Load back
const items = JSON.parse(localStorage.getItem('todos') || '[]');

To‑Do List (Working Example)

    How it works: The form listens for a submit event, creates a task object, stores tasks in localStorage, and renders the list. Each item supports complete/uncomplete via a checkbox and delete via a button.
